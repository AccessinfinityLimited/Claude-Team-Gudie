Install GitHub CLI & Authenticate

Homebrew → gh install → login

›
ℹ️ Run these commands one at a time in your Terminal (Mac) or Command Prompt (Windows).
Terminal 
# Step 1 — Install Homebrew (Mac only)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Step 2 — Install GitHub CLI
brew install gh

# Step 3 — Log in to GitHub
gh auth login
✅ When prompted by gh auth login — choose GitHub.com → HTTPS → Login with a web browser and follow the steps.

Daily Git & Firebase Workflow

Pull → push → deploy — do this every day

›
💡 Good habit: Always pull first → make changes → push → deploy. Never push without pulling first to avoid conflicts.

⬇ Pull Latest Code

Always start here — get the latest version from remote.

git pull origin main

⬆ Push Your Changes

Stage, commit, and push your work.

git add .
git commit -m "message"
git push origin main

🚀 Deploy to Firebase

Ship the latest code live after pushing.

firebase deploy

✅ Verify Everything Works

Run these commands in your terminal to confirm each tool is installed.

Tool Run This Expected Result
Git git --version Shows a version number
Claude Desktop Open the app Logged in successfully
GitHub gh auth status Shows "Logged in to github.com"
Firebase firebase --version Shows a version number
